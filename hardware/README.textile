h1. MobileHUD Hardware

This is the README file for the DroidHUB hardware directory.

This directory contains hardware design files for the hardware portion of the MobileHUD project.

* The hw-controller/ directory contains schematic and PCB layout files for the MobileHUD controller module.
* The hw-display/ directory contains schematic and PCB layout files for the MobileHUD display module.

Design documents for MobileHUD hardware can be found in the MobileHUD wiki's "Hardware section":https://github.com/davidrea/MobileHUD/wiki/Hardware-Design.
